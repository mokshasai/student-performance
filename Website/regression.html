Overview

Regression is a supervised learning technique used to predict a continuous numeric outcome, rather than a category. In this project, regression is applied to directly predict each student's final grade (G3) on a 0–20 scale. Unlike classification — which groups students into Low, Medium, or High buckets — regression provides a precise predicted score, making it useful for fine-grained interventions (e.g., predicting that a student will score 11.4 rather than simply "Medium"). The simplest approach is Ordinary Least Squares (OLS) Linear Regression, which fits a hyperplane through the feature space by minimizing the sum of squared residuals (the differences between predicted and actual values). Multiple regression includes all features simultaneously, allowing the model to estimate how each attribute (e.g., study time, absences) contributes to the final grade while holding all other factors constant — producing interpretable coefficients for each feature.

Beyond simple linear regression, Ridge and Lasso regularization are applied to prevent overfitting. Ridge regression adds a penalty term proportional to the squared magnitude of the model coefficients, shrinking all coefficients toward zero but never exactly to zero. Lasso regression uses an absolute value penalty, which has the useful property of forcing some coefficients to exactly zero — effectively performing automatic feature selection. This is particularly valuable in this dataset where many features are binary (one-hot encoded) and some may have near-zero predictive value. The best regression approach is selected based on cross-validated RMSE (Root Mean Square Error) and R² (coefficient of determination), which measures how much of the variance in grades the model explains.

📈

Key Metrics: RMSE (lower = better, measures average prediction error in grade points), R² (higher = better, proportion of variance explained — 1.0 = perfect), MAE (Mean Absolute Error — average absolute difference between predicted and actual grades).

Data

Regression uses the cleaned, encoded dataset with G3 as the numeric target variable (not a label). G1 and G2 are excluded to measure how well the model predicts final grades from behavioral and demographic features alone. All features are normalized using StandardScaler for consistency across regularized models.

Raw data — 395 students with 33 features including the G3 numeric target (0–20 scale).
Cleaned data — fully numeric after encoding; G3 retained as continuous regression target.
